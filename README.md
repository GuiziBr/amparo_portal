# Amparo Portal

Portal para gerenciamento de atividades e pacientes

## Tecnologias

- ReactJs
- Typescript

## Guia de inicialização

### Pré-requisitos
- NodeJs 12
- yarn/npm

1. Faça o clone do projeto
2. Execute o comando yarn ou npm install, para instalar as dependências do projeto
2. Execute o comando "yarn start" ou "npm start"
